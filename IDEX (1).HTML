Gestión de Recursos Educativos

La aplicación web y móvil de la Biblioteca Técnico Palermo permite administrar y consultar todos los recursos educativos del colegio. Fue diseñada para facilitar el acceso a los materiales digitales y físicos, y mejorar la comunicación entre estudiantes, docentes y bibliotecarios.

Gracias a esta herramienta, los usuarios pueden buscar libros, reservar material, consultar disponibilidad y recibir notificaciones. El sistema utiliza tecnologías modernas como HTML, CSS y bases de datos para garantizar una experiencia ágil y segura.

Recursos disponibles

  • Libros académicos
  • Revistas digitales
  • Material audiovisual
  • Guías de estudio
  • Documentos institucionales

Equipo de desarrollo

  • Estudiantes del programa técnico
  • Área de tecnología e informática
  • Bibliotecarios del colegio
